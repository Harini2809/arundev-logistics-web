## arundev-logistics-web

### Prerequisites

- Node.js
- pnpm installed globally

or

```bash
npm install -g pnpm
```

### Install deps

From the project root:

```bash
pnpm install
```

### Run the dev server

```bash
pnpm run dev
```

http://localhost:5173 Open it in your browser.

> push to main branch to deploy the changes to production site https://arundevlogistics.com/
